**Bugfix**

- `kolt add` no longer fails on transitives whose POMs use `${project.groupId}` / `${project.version}` and inherit those coordinates from `<parent>`. Affected `kolt add io.ktor:ktor-server-netty-jvm` and any chain pulling `io.netty:netty-codec-http2:4.2.12.Final` (#421).

**Installation**

```sh
curl -fsSL https://raw.githubusercontent.com/snicmakino/kolt/main/install.sh | sh
```
